# Smart Calculator

A terminal-based interactive calculator which supports the following:
- Basic arithmetic: addition, subtraction, multiplication and division,
- The use of parentheses,
- Variables.

### Example
```
a = 2
b = 10
a * b - 7
13.0
a * (b - 7)
6.0
6 + 370 / 85 - 8
2.352941176470587
/exit
Bye! 
```
